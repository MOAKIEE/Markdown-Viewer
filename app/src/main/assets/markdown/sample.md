# 欢迎使用 Markdown Viewer

这是一个简单的 Markdown 预览应用。

## 功能特点

- 浏览手机存储中的 Markdown 文件
- 支持 `.md` 和 `.markdown` 扩展名
- 渲染 Markdown 格式，包括：
  - 标题 (H1-H6)
  - 列表（有序和无序）
  - **粗体** 和 *斜体*
  - [链接](https://example.com)
  - 表格
  - 代码块

## 示例内容

### 列表示例

1. 第一项
2. 第二项
3. 第三项

- 项目 A
- 项目 B
- 项目 C

### 代码示例

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Markdown!");
    }
}
```

### 表格示例

| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 28   | 北京 |
| 李四 | 32   | 上海 |
| 王五 | 25   | 广州 |

> 提示：点击文件列表中的 Markdown 文件即可查看渲染效果。
